# SpamApp-Windows
Tüm uygulamalarda çalışan hotkeyli Spam uygulaması.
Uygulama Visual Studio kullanılarak, C# Windows Forms dilinde geliştirildi. 

### Kullanım:
Hotkey : F6
Spam Limiti ve ms süresini ayarlayarak spam atabilirsiniz.
